# THE AR WARS - 動作確認ガイド

このドキュメントでは、全機能の動作確認手順を説明します。

## 📋 事前準備

### 1. DBマイグレーションの確認

```bash
# 開発サーバーを起動
npm run dev
```

1. ブラウザで `http://localhost:3000/admin` を開く
2. 右上の **「🔍 DB状態チェック」** ボタンをクリック
3. 以下のいずれかが表示されます：

**✅ 正常な場合:**
```
✅ データベースは正常です！

全ての機能が使用可能です。
```

**❌ 問題がある場合:**
```
⚠️ データベースに問題があります

❌ Missing debt_count column in teams table
   SQL: ALTER TABLE teams ADD COLUMN IF NOT EXISTS debt_count INT NOT NULL DEFAULT 0;

❌ Missing ends_at column in game_state table
   SQL: ALTER TABLE game_state ADD COLUMN IF NOT EXISTS ends_at TIMESTAMP WITH TIME ZONE;
```

問題がある場合は、Supabase SQL Editorで表示されたSQLを実行してください。

### 1. UIコンポーネントのテスト

#### ✅ Hydration Errorの確認
1. ブラウザのコンソール (F12) を開く
2. 管理画面で **「+ チーム追加」** ボタンをクリック

- チーム名を入力して追加できる

### 2. 融資機能のテスト

   - 借金回数が **1** になっている

   - **「💀 債務者」** バッジが表示されている

### 3. タイマー機能のテスト

#### ✅ タイマー設定の確認

**準備:**
1. チーム画面を別タブで開く: `/team/[team_id]` （URLは管理画面でコピー）
2. メインスクリーンも別タブで開く: `/screen`
3. ブラウザのコンソール (F12) を開く

**テスト手順:**

1. **30秒タイマーを設定**
   - 管理画面で **「30秒」** ボタンをクリック
   - 成功アラートを確認：
     ```
     ✅ タイマー開始: 30秒
     終了時刻: 14:30:45
     ```
   - コンソールで確認：
     ```
     [Timer] Starting timer: 30 seconds
     [Timer] Calculated end time: { now: "...", endsAt: "...", seconds: 30 }
     [Timer] Timer set successfully: { ... }
     ```

2. **全画面でカウントダウンを確認**
   - チーム画面に **「残り時間」** カードが表示される
   - メインスクリーンにも **「残り時間」** が表示される
   - 両方で同じ時間が表示される（±1秒の誤差は許容）

3. **色の変化を確認**
   - 30秒以上: **緑色** (border-green-500)
   - 11-30秒: **黄色** (border-yellow-500)
   - 1-10秒: **赤色 + 点滅** (border-red-500 + animate-pulse)

4. **時間切れの動作確認**
   - 0秒になると **「⏱️ 時間切れ！」** が表示される
   - チーム画面で以下を確認：
     - ベット/入札の入力欄が **Disabled** になる
     - ボタンが **グレーアウト** する
     - ボタンのテキストが **「時間切れ」** になる
   - コンソールで確認：
     ```
     [CountdownTimer] Timer expired, calling onExpire
     [Team] Timer expired - locking controls
     ```

5. **タイマークリアの確認**
   - 管理画面で **「タイマークリア」** ボタンをクリック
   - 成功アラート: `✅ タイマークリア完了`
   - チーム画面で以下を確認：
     - カウントダウン表示が**消える**
     - 入力欄とボタンが**有効**になる
   - コンソールで確認：
     ```
     [Timer] Clearing timer
     [Timer] Timer cleared successfully
     [Team] Timer cleared - unlocking controls
     ```

6. **新しいタイマーでリセット確認**
   - タイマーが0秒の状態（または時間切れ後）
   - 管理画面で **「60秒」** ボタンをクリック
   - チーム画面で以下を確認：
     - カウントダウンが **60** から開始
     - 入力欄とボタンが**有効**になる
   - コンソールで確認：
     ```
     [Team] Timer check: { ends_at: "...", remaining: 60, is_expired: false }
     ```

## 🐛 トラブルシューティング

### エラーが出る場合

1. **ブラウザのコンソールを開く** (F12)
2. **該当する操作を実行**
3. **エラーログを確認**:
   - `[Loan]` - 融資関連
   - `[Timer]` - タイマー関連
   - `[CountdownTimer]` - カウントダウン表示
   - `[Team]` - チーム画面

### よくある問題

#### Q1: 融資ボタンを押してもエラーが出る
**A:** DB状態チェックで `debt_count` カラムを確認してください。

#### Q2: タイマーが表示されない
**A:** DB状態チェックで `ends_at` カラムを確認してください。

#### Q3: タイマーが0になっても操作がロックされない
**A:** コンソールで以下を確認：
```
[CountdownTimer] Timer expired, calling onExpire
[Team] Timer expired - locking controls
```
これらのログが出ていれば正常です。ページをリロードしてみてください。

#### Q4: サウンドが鳴らない
**A:** 以下を確認：
- `/public/sounds/` にMP3ファイルがあるか
- ファイル名が正しいか（drumroll.mp3など）
- ブラウザの音量設定
