# サウンドファイル

このディレクトリに以下のサウンドファイル（MP3形式）を配置してください：

## 必要なファイル

- `drumroll.mp3` - ドラムロール（盛り上がり演出用）
- `fanfare.mp3` - ジャジャーン（正解・確定演出用）
- `gavel.mp3` - カーン（オークション落札音）
- `sad.mp3` - ガーン（没収・失敗演出用）

## サウンドファイルの入手方法

以下のようなサイトからフリー素材をダウンロードできます：

- **効果音ラボ**: https://soundeffect-lab.info/
- **魔王魂**: https://maou.audio/
- **DOVA-SYNDROME**: https://dova-s.jp/

各ファイルは上記のファイル名で保存してください。

## 使用方法

管理画面の「BGM・SE コントロール」セクションからボタンをクリックすると、メインスクリーンで該当するサウンドが再生されます。
