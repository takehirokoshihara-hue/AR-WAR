# Game 3 オークション機能 - 運用ガイド

Game 3は**シンプルなオークション機能**に変更されました。効果の自動計算は廃止し、純粋な入札機能のみを提供します。

---

## 🎯 新しいGame 3の仕様

### 概要
- **3回のオークション**を実施
- 各回の**最高入札者**が「封筒」を獲得
- 封筒の中身（効果）は**開封まで不明**
- 効果の適用は**管理者が手動**で行う

### システムの役割
✅ **システムがやること:**
- 入札額の記録
- 最高入札額・入札者の表示
- 入札確定時のAR減額（入札額分のみ）
- 入札のリセット

❌ **システムがやらないこと:**
- 効果の自動計算（倍増、半減など）
- チーム間のAR移動
- 複雑なルール処理

---

## 📋 運用フロー（3ラウンド）

### ラウンド1

1. **管理画面でGame 3フェーズに切り替え**
   - `/admin` → 「Game 3」ボタン

2. **各チームが入札**
   - チーム画面で入札額を入力
   - 現在の最高額より高い額を入力
   - 「ALL IN」ボタンで全額入札も可能

3. **管理画面で確認**
   - 「Game 3: オークション管理」セクション
   - 現在の最高入札者と入札額が表示される

4. **入札締切（タイマー使用推奨）**
   - 「⏱️ カウントダウンタイマー」で30秒などを設定
   - 時間切れで自動的に入札不可になる

5. **入札確定**
   - 「✅ 入札確定」ボタンをクリック
   - 確認ダイアログで「OK」
   - **最高入札者のARから入札額が減額される**
   - 成功メッセージで落札者名と金額を確認

6. **リアルの封筒を渡す**
   - 落札者に封筒（番号1）を渡す
   - **中身は開封しない**

7. **リセット**
   - 「🔄 入札リセット」ボタンをクリック
   - 次のラウンド準備完了

### ラウンド2・ラウンド3

- 同じ手順を繰り返す
- 封筒（番号2）、封筒（番号3）を渡す

---

## 🎁 封筒開示フェーズ

### 1. 一斉開封
- 3人の落札者が**同時に封筒を開封**
- カードの効果を発表

### 2. 効果の例
```
封筒1: +200万AR
封筒2: チームAとBのAR残高を交換
封筒3: AR残高が半分になる
```

### 3. 管理者が手動でAR調整

#### 方法1: AR残高の直接編集
1. 管理画面のチーム管理テーブルへ
2. 変更したいチームのAR残高をクリック
3. 新しい数値を入力
4. `Enter`キーまたは`✓`ボタンで保存

**例：**
- チームA（現在: 1,500,000 AR）
- 封筒の効果: +200万AR
- クリック → `3500000` と入力 → Enter
- 確認ダイアログで「OK」

#### 方法2: 計算してから入力
- 現在の残高を確認
- 電卓で計算
- 新しい残高を入力

**例：チーム間のAR交換**
```
チームA: 1,500,000 AR
チームB: 2,300,000 AR

↓ 交換 ↓

チームA: クリック → 2300000 と入力
チームB: クリック → 1500000 と入力
```

## 🖥️ 管理画面の使い方

### Game 3: オークション管理セクション

```
┌─────────────────────────────────┐
│ 現在の最高入札                    │
│ 入札者: abc-123-def-456          │
│ 入札額: 1,500,000 AR            │
└─────────────────────────────────┘

[✅ 入札確定]  [🔄 入札リセット]
```

#### ボタンの役割

**✅ 入札確定:**
- 最高入札者のARから入札額を減額
- 落札者情報をアラート表示
- リアルの封筒を渡すタイミング

**🔄 入札リセット:**
- 入札情報をクリア
- 次のラウンドの準備
- 金額は0に、入札者はなしに戻る

### チーム管理テーブル - AR編集

```
┌────┬──────┬──────────────┬────┐
│順位│ チーム │  AR残高 ✎    │操作│
├────┼──────┼──────────────┼────┤
│ 1  │Team A│ 1,500,000 AR │... │
│    │      │  ↑クリック可能  │    │
└────┴──────┴──────────────┴────┘
```

#### AR編集の手順
1. AR残高の数字をクリック（`✎`アイコンが表示される）
2. 入力欄が表示される
3. 新しい数値を入力
4. `Enter`または`✓`で保存
5. `Escape`または`✕`でキャンセル

## 💡 運用のコツ

### タイミング管理
- **タイマー機能を活用**
  - 各ラウンド60秒など
  - 時間切れで自動ロック
  - 公平性を保てる

### 入札確認
- **入札確定の前に確認**
  - 最高入札者が正しいか
  - 金額が妥当か
  - チームのAR残高は足りているか

### AR調整
- **スクリーンを見ながら調整**
  - `/screen` で全員の残高を表示
  - 変更がリアルタイムで反映
  - 間違えてもすぐ修正可能

### 記録
- **各ラウンドの結果をメモ**
  ```
  ラウンド1: Team A - 1,200,000 AR → 封筒1
  ラウンド2: Team C - 800,000 AR  → 封筒2
  ラウンド3: Team B - 1,500,000 AR → 封筒3
  ```
- 封筒開示後の調整に便利

## ⚠️ 注意事項

### 1. 入札確定は慎重に
- ✅ 入札確定をクリックすると**即座にARが減額**される
- 元に戻す機能はない
- AR編集で手動で戻す必要がある

### 2. リセットのタイミング
- 入札確定→封筒を渡す→リセット の順番を守る
- リセット後は入札情報が消える

### 3. AR編集は自由だが慎重に
- どんな数値にも変更可能
- 間違えた場合は再度クリックして修正
- 大きな変更は確認ダイアログが出る

### 4. ネットワーク接続
- リアルタイム同期のため安定した接続が必要
- WiFiの状態を事前確認

## 🔍 トラブルシューティング

### Q: 入札確定ボタンが押せない
**A:** 以下を確認:
- 入札があるか（誰かが入札したか）
- 管理画面がローディング中でないか
- ブラウザのコンソールにエラーがないか

### Q: AR編集が保存されない
**A:** 以下を試す:
- 数値が有効か（マイナスは不可）
- ネットワーク接続を確認
- ブラウザのコンソールでエラー確認
- ページをリロード

### Q: 間違えてARを変更してしまった
**A:** 落ち着いて:
1. すぐに再度クリック
2. 正しい数値を入力
3. 保存
4. メインスクリーンで確認

### Q: リセット後も入札が表示される
**A:** 以下を確認:
- ページをリロード
- コンソールにエラーがないか
- もう一度リセットボタンを押す
